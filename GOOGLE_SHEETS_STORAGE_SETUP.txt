Google Sheets permanent storage setup for Streamlit Cloud

1. Create a Google Cloud service account.
2. Enable the Google Sheets API and Google Drive API in that Google Cloud project.
3. Create a JSON key for the service account.
4. Create a Google Sheet for this app.
5. Share the Google Sheet with the service account email as Editor.
6. Copy the spreadsheet ID from the Google Sheet URL.
   Example URL:
   https://docs.google.com/spreadsheets/d/SPREADSHEET_ID_HERE/edit
7. In Streamlit Cloud, open your app.
8. Go to Manage app -> Settings -> Secrets.
9. Add secrets using this format:

google_sheet_id = "YOUR_SPREADSHEET_ID"

[gcp_service_account]
type = "service_account"
project_id = "..."
private_key_id = "..."
private_key = "-----BEGIN PRIVATE KEY-----\n...\n-----END PRIVATE KEY-----\n"
client_email = "your-service-account@your-project.iam.gserviceaccount.com"
client_id = "..."
auth_uri = "https://accounts.google.com/o/oauth2/auth"
token_uri = "https://oauth2.googleapis.com/token"
auth_provider_x509_cert_url = "https://www.googleapis.com/oauth2/v1/certs"
client_x509_cert_url = "..."
universe_domain = "googleapis.com"

10. Commit/push goal_app.py and requirements.txt.
11. Reboot the Streamlit app.

The app will create a worksheet tab named goal_progress automatically.
If secrets are missing or access fails, it falls back to local JSON and shows a warning in the app.
